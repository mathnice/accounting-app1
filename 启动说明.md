# 跨端记账应用 - 启动说明

## 当前状态

应用已成功启动！

- 后端服务: http://localhost:3000
- 网页端: http://localhost:5173

## 使用方法

1. 打开浏览器访问 http://localhost:5173
2. 首次使用需要注册账号（输入手机号、验证码、密码）
3. 验证码功能为模拟，任意输入即可
4. 登录后可以开始记账

## 手动启动步骤（如果服务停止）

### 启动后端
```bash
cd accounting-app/backend
npm install
npm run dev
```

### 启动网页端
```bash
cd accounting-app/web
npm install
npm run dev
```

## 技术说明

- 后端使用 sql.js（纯JavaScript SQLite），无需安装额外软件
- 数据文件存储在 `backend/data/accounting.db`
- 所有数据都保存在本地

## 常见问题

**Q: 端口被占用？**
A: 修改 `backend/src/config/index.ts` 中的端口号

**Q: 网页打不开？**
A: 确保后端和网页端服务都已启动
